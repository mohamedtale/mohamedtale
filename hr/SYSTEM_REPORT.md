# تقرير تفصيلي شامل
# نظام إدارة شؤون الموظفين
# الجهاز التنفيذي لحفر وصيانة آبار المياه

---

## معلومات المشروع
- **الجهة:** الجهاز التنفيذي لحفر وصيانة آبار المياه
- **النظام:** نظام إدارة شؤون الموظفين
- **الاستخدام:** داخلي فقط (شبكة الجهاز)
- **المستخدمون:** 4 موظفين في قسم شؤون الموظفين
- **التقنيات:** Next.js + Express.js + PostgreSQL + Docker

### 2. نظام الإجازات

#### أنواع الإجازات وقواعدها:
| النوع | المدة | قواعد خاصة |
|-------|-------|------------|
| سنوية | رصيد خاص بكل موظف | قواعد خصم مختلفة حسب نوع الموظف |
| مرضية | 60 يوم / سنة | بعد 60 يوم → تنبيه إحالة للجنة طبية |
| طارئة | 12 يوم / سنة | |
| أمومة | بدون قيود | |
| حج | شهر واحد | مرة واحدة في العمر فقط |
| بدون راتب | شهرين → سنة | تُحفظ في الملف الشخصي |

#### قواعد خصم الإجازة السنوية:
| نوع الموظف | قاعدة الخصم |
|-----------|-------------|
| إداري | لا يُخصم الجمعة والسبت + أيام العطل الرسمية |
| غفارة | تُخصم جميع الأيام بدون استثناء، اليوم الواحد = 3 أيام من الرصيد |

#### شاشة إدارة العطل الرسمية:
- يدخل المدير تواريخ العطل في بداية كل سنة
- النظام يستثنيها تلقائياً عند حساب إجازات الموظفين الإداريين

### 3. نظام الحضور والانصراف

#### مصدر البيانات:
- استيراد ملف `.dat` مباشرة من جهاز ZKTeco
- أول بصمة في اليوم = دخول / آخر بصمة = خروج

#### إعدادات الدوام (تُدخل يدوياً):
- وقت الدخول الرسمي: 08:30
- وقت الخروج الرسمي: 14:30
- قابل للتغيير في أي وقت

#### واجهة الحضور الشبكية:
| الحالة | الرمز | اللون |
|--------|-------|-------|
| حضور منتظم | ✅ | أخضر |
| تأخر | 🕐 | أصفر |
| خروج مبكر | 🚪 | برتقالي |
| غياب | ❌ | أحمر |
| إجازة | 🏖️ | أزرق |
| عطلة رسمية | ⚪ | رمادي |
| غفارة | 🌙 | بنفسجي |
| تكليف | 📋 | سماوي |
| معدّل يدوياً | ✏️ | ذهبي |

#### التعديل اليدوي:
- عند الضغط على أي خلية → نافذة تعديل
- تختار الحالة الصحيحة + سبب التعديل
- كل تعديل يُسجل تلقائياً (من غيّر / متى / لماذا)

#### تقرير الموظف الشهري:
- جدول تفصيلي بكل أيام الشهر
- دخول / خروج / ساعات عمل / تأخر / خروج مبكر / الحالة
- ملخص: حضور / غياب / تأخر / ساعات فعلية / نسبة حضور %
- تصدير: طباعة / PDF / Excel

#### تنبيهات ذكية:
- غياب أكثر من 3 أيام متتالية → تنبيه فوري
- تأخر أكثر من 5 مرات في الشهر → تنبيه

### 4. التكاليف والفحص الطبي

#### شاشة تكليف المهام:
- الرقم الوظيفي → يجلب الاسم تلقائياً
- تاريخ البداية + تاريخ النهاية
- رقم المرجع / القرار + ملاحظات
- يظهر في شاشة الحضور تلقائياً بأيام التكليف

#### شاشة الفحص الطبي:
- الرقم الوظيفي → يجلب الاسم والرقم الوطني تلقائياً
- اسم الطبيب + المستشفى / العيادة
- التشخيص + تاريخ الفحص
- مدة الراحة → تُخصم من رصيد المرضية تلقائياً
- عند وصول المرضية 60 يوم → تنبيه إحالة للجنة الطبية

---

### 5. أذونات الخروج

#### البيانات:
- الرقم الوظيفي → يجلب الاسم تلقائياً
- نوع الإذن: شخصي / عمل / طبي / طارئ
- تاريخ الخروج
- وقت الخروج + وقت العودة → يحسب المدة بالدقائق تلقائياً
- سبب الخروج

#### القواعد:
- كل موظف له **2 إذن فقط في الشهر**
- النظام يمنع التسجيل عند تجاوز الـ 2 ويُنبه
- لا تأثير على الراتب أو الإجازة
- تقرير شهري بإجمالي أذونات كل موظف

### 6. نظام العلاوات والترقيات

#### قواعد العلاوة:
- علاوة واحدة كل سنة لكل موظف
- الدرجات الوظيفية من 3 إلى 16

#### عدد العلاوات للترقية:
| الدرجة | علاوات مطلوبة للترقية |
|--------|----------------------|
| 3 → 9 | 4 علاوات |
| 10 | 5 علاوات |
| 11 → 16 | 4 علاوات |

#### نظام المربوط:
- ليس لجميع الموظفين
- عند إدخال بيانات الموظف يُختار: عادي أم مربوط
- إذا مربوط → يُدخل رقم المربوط (= رقم العلاوة الحالية داخل الدرجة)
- مثال: درجة 8 مربوط 4 → يحتاج 4 علاوات إضافية للترقية لدرجة 9

#### تقرير العلاوات المستحقة:
| الاسم | الرقم الوظيفي | الدرجة/العلاوة السابقة | التاريخ | الدرجة/العلاوة الحالية | التاريخ | مرشح للترقية | ملاحظات |
|-------|--------------|----------------------|---------|----------------------|---------|--------------|---------|

#### ما يفعله النظام تلقائياً:
- ينبه عند استحقاق كل علاوة سنوية
- ينبه عند اكتمال شروط الترقية
- يُصدر كشف رسمي للمرشحين (شؤون الموظفين فقط - بدون علاقة بالراتب)
- يحفظ تاريخ كل علاوة وترقية في ملف الموظف

### 8. نظام المراسلات

#### أنواع المراسلات:
| الرمز | النوع |
|-------|-------|
| 📥 | بريد وارد |
| 📤 | بريد صادر |
| 🏖️ | إجازات |
| 📋 | طلبات |
| 📜 | قرارات داخلية |
| 📜 | قرارات خارجية |
| ✍️ | رسائل مهمش عليها |
| 🔖 | تأشيرات |

#### بيانات كل مراسلة:
- رقم الخطاب (يُدخل يدوياً من المحفوظات)
- تاريخ الخطاب
- الجهة المرسلة / المستقبلة
- الموضوع
- اسم الموظف المعني (إن وجد)
- المرفقات (صورة من الماسح الضوئي)

#### نظام تدفق العمل (Workflow):
مثال: طلب سلفة مالية:
```
موظف يقدم الطلب
      ↓
المدير → يهمش "أحيل للقانوني"
      ↓
القانوني → يهمش "لا مانع"
      ↓
المدير → يهمش "أحيل للمالية للإجراء"
      ↓
المالية → تنفذ وتغلق الطلب
```
- كل شخص يرى فقط ما وصله
- تنبيه فوري عند وصول الطلب
- سجل كامل لكل تهميش (من / متى / ماذا قال)
- تنبيه إذا توقف الطلب أكثر من X أيام
- خاص بشؤون الموظفين فقط

#### حالات المراسلة:
جديد → قيد المعالجة → تم الرد → مغلق

- بحث بالاسم أو الرقم الوطني أو الرقم الوظيفي

### 10. كشف العاملين

#### حقول الكشف:
1. اسم الموظف
2. الرقم الوظيفي
3. الرقم الوطني
4. المؤهل الدراسي
5. تاريخ التعيين
6. تاريخ المباشرة
7. المسمى الوظيفي
8. الإدارة / القسم / المكتب
9. الدرجة والعلاوة الحالية

#### مميزات الكشف:
- **قوالب متعددة** لكل جهة (وزارة / مدير عام / مالية...)
- **فلترة مرنة:** إدارة / درجة / نوع موظف / نوع تعاقد / حالة / تاريخ تعيين / مرشحون للترقية
- **ترتيب مرن:** رقم وظيفي تصاعدي أو تنازلي / أبجدي / تاريخ تعيين / حسب الإدارة
- **معاينة قبل الطباعة**
- **تصدير:** طباعة / PDF / Excel / Word
- **كشف مقارن** بين السنوات
- **كشف التغييرات الشهرية**
- **حفظ الكشوفات السابقة** تلقائياً بتاريخها

#### شكل الكشف:
- رأس الصفحة: شعار الجهاز + تاريخ الإصدار (ثابت تلقائي)
- أسفل الكشف: حقل توقيع قابل للإضافة والإزالة
- إجمالي عدد الموظفين في الأسفل

### 11. التقارير

#### أ. تقارير الموظفين:

**1. تقرير المرشحين للترقية:**
الاسم / الرقم / الدرجة الحالية / عدد العلاوات / تاريخ الاستحقاق

**2. تقرير المقبلين على التقاعد:**
الاسم / الرقم / تاريخ الميلاد / الجنس / سن التقاعد / المدة المتبقية
اختيار الفترة: 6 أشهر / سنة / سنتين

**3. تقرير من تنتهي عقودهم:**
الاسم / الرقم / نوع العقد / تاريخ الانتهاء / المدة المتبقية
اختيار الفترة: 30 / 60 / 90 يوم

**4. تقرير الموظفين حسب الإدارة (متطور):**
إحصائيات كل إدارة / مقارنة بين الإدارات / رسم بياني

#### ب. تقارير الإجازات:

**5. تقرير رصيد الإجازات:**
الرصيد الأصلي / المستخدم / المتبقي لكل موظف

**6. تقرير الموظفين في إجازة حالياً:**
النوع / تاريخ البداية والنهاية / تاريخ العودة المتوقع

#### ج. تقارير العلاوات:

**7. تقرير العلاوات المستحقة هذا الشهر:**
الدرجة والعلاوة السابقة والحالية / مرشح للترقية / ملاحظات

**8. تقرير تاريخ العلاوات لكل موظف:**
كامل تاريخ العلاوات والترقيات

#### د. التقارير الإدارية:

**9. التقرير الشهري للمدير:**
ملخص شامل لكل ما حدث في الشهر

**10. التقرير السنوي للوزارة:**
5 أقسام: موظفون / إجازات / علاوات وترقيات / تقاعد وفصل ونقل / مراسلات

**11. التقرير الربع سنوي (الأهم والجوهر):**
يُرسل لـ: مدير الشؤون الإدارية / مدير مكتب المراجعة الداخلية / مدير عام الجهاز

| القسم | المحتوى |
|-------|---------|
| الأول | إحصائيات الموظفين (توزيع / تغييرات / ذكور وإناث) |
| الثاني | الإجازات بأنواعها وأيامها |
| الثالث | العلاوات والترقيات |
| الرابع | التقاعد والفصل والنقل |
| الخامس | المراسلات (وارد/صادر/طلبات/قرارات) مع مقارنة بالربع السابق |

- قابل للتخصيص (إضافة / حذف أقسام)
- تنبيه تلقائي عند اقتراب موعد كل ربع
- مقارنة ذكية مع الربع السابق
- تصدير: طباعة / PDF / Word / Excel

#### مميزات جميع التقارير:
- قابلة للتخصيص
- حفظ قوالب لكل جهة
- حفظ نسخة من كل تقرير في الأرشيف تلقائياً

### 12. نظام الصلاحيات

#### المستويات:
| المستوى | المسمى | الصلاحيات |
|---------|--------|-----------|
| 1 | مدير النظام | كل شيء بدون قيود |
| 2 | مدير القسم | كل شيء ما عدا إعدادات النظام |
| 3 | موظف أول | إضافة وتعديل بدون حذف |
| 4 | موظف | إضافة فقط |
| 5 | قراءة فقط | عرض بدون أي تعديل |

#### صلاحيات الشاشات:
- كل شاشة يمكن تفعيلها أو تعطيلها لكل موظف بشكل مستقل
- مثال: موظف A يرى الإجازات والموظفين فقط - لا يرى الحضور والتقارير

#### الأمان:
- سجل تدقيق كامل (من دخل / ماذا فعل / متى)
- تنبيه عند محاولة دخول غير مصرح
- قفل الحساب بعد 3 محاولات فاشلة

### 13. لوحة التحكم الرئيسية

#### الشريط العلوي:
- إجمالي الموظفين
- في إجازة حالياً
- طلبات معلقة
- تنبيهات جديدة

#### التنبيهات الذكية:
- من يستحق علاوة هذا الشهر

- طلبات إجازة تنتظر الموافقة
- مراسلات معلقة لم يُرد عليها
- موظفون وصلوا لـ 50 يوم مرضية

#### الإجازات:
- الموظفون في إجازة حالياً مع تاريخ العودة
- من سيعود هذا الأسبوع
- من سيبدأ إجازة هذا الأسبوع

#### الرسوم البيانية:
- توزيع الموظفين حسب الإدارة
- نسبة أنواع التعاقد
- مقارنة الإجازات هذا الشهر مع الشهر الماضي

- آخر 10 عمليات تمت في النظام

#### التقويم الشهري:
- أيام العطل الرسمية
- من في إجازة في كل يوم
- مواعيد استحقاق العلاوات
