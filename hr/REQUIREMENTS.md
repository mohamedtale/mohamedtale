## ثانياً: نظام الإجازات

### أنواع الإجازات:
| النوع | المدة | قواعد خاصة |
|-------|-------|------------|
| سنوية | رصيد خاص بكل موظف | |
| مرضية | 60 يوم/سنة | بعد 60 يوم → تنبيه إحالة للجنة طبية |
| طارئة | 12 يوم/سنة | |
| أمومة | بدون قيود | |
| حج | شهر | مرة واحدة في العمر |
| بدون راتب | شهرين → سنة | تُحفظ في الملف الشخصي |

### قواعد الخصم:
- **موظف إداري:** لا يُخصم الجمعة والسبت + أيام العطل الرسمية
- **موظف غفارة:** تُخصم جميع الأيام، اليوم الواحد = 3 أيام من الرصيد

### شاشة إدارة العطل الرسمية:
- يدخل المدير تواريخ العطل في بداية كل سنة
- النظام يستثنيها تلقائياً عند حساب الإجازات الإدارية

## ثالثاً: نظام الحضور والانصراف

### طريقة العمل:
- استيراد ملف `.dat` مباشرة من جهاز ZKTeco
- أول بصمة في اليوم = دخول
- آخر بصمة في اليوم = خروج
- حساب ساعات العمل تلقائياً

### إعدادات الدوام:
- وقت الدخول الرسمي: يُدخل يدوياً (حالياً 08:30)
- وقت الخروج الرسمي: يُدخل يدوياً (حالياً 14:30)
- قابل للتغيير في أي وقت (صيف/شتاء)

### واجهة الحضور:
- عرض شبكي (كل موظف × كل يوم في الشهر)
- ألوان وحالات:
  - ✅ حضور منتظم
  - 🕐 تأخر
  - 🚪 خروج مبكر
  - ❌ غياب
  - 🏖️ إجازة
  - ⚪ عطلة
  - 🌙 غفارة
  - 📋 تكليف
- عند الضغط على أي خلية → نافذة تعديل يدوي مع سبب التعديل
- كل تعديل يدوي يُسجل في السجل التاريخي (من غيّر، متى، لماذا)
- الخلية المعدلة تظهر علامة ✏️
- ربط الحضور بالإجازات تلقائياً

### إحصائيات الأعلى:
- غياب / تأخر / انصراف مبكر / حضور منتظم / إجمالي القوة / نسبة الحضور %

### تقرير الموظف الشهري:
- جدول تفصيلي بكل أيام الشهر
- دخول / خروج / ساعات العمل / تأخر / خروج مبكر / الحالة
- ملخص: إجمالي حضور، غياب، تأخر، دقائق تأخر، ساعات فعلية، نسبة حضور
- تصدير: طباعة / PDF / Excel

### تنبيهات ذكية:
- غياب أكثر من 3 أيام متتالية → تنبيه
- تأخر أكثر من 5 مرات في الشهر → تنبيه

## رابعاً: التكاليف والفحص الطبي

### شاشة تكليف المهام:
- الرقم الوظيفي ← يجلب الاسم تلقائياً
- تاريخ البداية + تاريخ النهاية
- رقم المرجع / القرار
- ملاحظات
- يظهر في شاشة الحضور تلقائياً

### شاشة الفحص الطبي:
- الرقم الوظيفي ← يجلب الاسم والرقم الوطني تلقائياً
- اسم الطبيب
- اسم المستشفى / العيادة
- التشخيص
- تاريخ الفحص
- مدة الراحة (عدد الأيام) ← تُخصم من رصيد المرضية تلقائياً
- عند وصول المرضية 60 يوم → تنبيه "يجب الإحالة للجنة الطبية"

## خامساً: أذونات الخروج

- الرقم الوظيفي ← يجلب الاسم تلقائياً

## سادساً: نظام العلاوات والترقيات

### قواعد العلاوات:
- علاوة واحدة كل سنة
- الدرجات من 3 إلى 16

### عدد العلاوات للترقية:
| الدرجة | علاوات للترقية |
|--------|---------------|
| 3 → 9 | 4 علاوات |
| 10 | 5 علاوات |
| 11 → 16 | 4 علاوات |

### نظام المربوط:
- ليس لجميع الموظفين
- في بيانات الموظف يُختار: عادي أم مربوط
- إذا مربوط → يُدخل رقم المربوط
- المربوط = رقم العلاوة الحالية داخل الدرجة
- مثال: درجة 8 مربوط 4 → يحتاج 4 علاوات للترقية لدرجة 9

### تقرير العلاوات المستحقة:
| الاسم | الرقم الوظيفي | الدرجة والعلاوة السابقة | التاريخ | الدرجة والعلاوة الحالية | التاريخ | مرشح للترقية | ملاحظات |
|-------|--------------|------------------------|---------|------------------------|---------|--------------|---------|

### ما يفعله النظام:
- يتابع عدد العلاوات تلقائياً
- ينبه عند استحقاق العلاوة السنوية
- ينبه عند اكتمال شروط الترقية
- يُصدر كشف رسمي بالمرشحين للترقية (شؤون الموظفين فقط، بدون علاقة بالراتب)
- يحفظ تاريخ كل علاوة وترقية في ملف الموظف

## سابعاً: نظام التقاعد والفصل والنقل

### التقاعد:
- سن التقاعد الإجباري: الرجال 65 سنة / النساء 60 سنة
- التقاعد الاختياري: من سن 55 سنة
- عند بلوغ 55 سنة → تنبيه تلقائي لاستيفاء جميع البيانات

### إنهاء الخدمة وأنواعها:
| العلامة | السبب |
|---------|-------|
| 🔵 | تقاعد إجباري |
| 🟢 | تقاعد اختياري |
| 🟡 | انتهاء عقد |
| 🔴 | استقالة اعتبارية |
| 🟠 | استقالة اختيارية |
| ⚪ | انتهاء ندب |
| 🔄 | نقل لجهة أخرى |

- جميع المنتهية خدمتهم يُحفظون في أرشيف المتقاعدين مع علامة السبب

### النقل:
- **نقل خارجي وارد:** موظف يأتي من جهة أخرى → يُضاف للنظام
- **نقل خارجي صادر:** موظف يذهب لجهة أخرى → يُحذف من النظام + ملفه في الأرشيف بعلامة 🔄
- **نقل داخلي:** بين الإدارات داخل الجهاز → يُحفظ في سجل تاريخ الموظف

## ثامناً: نظام المراسلات

### أنواع المراسلات:
| النوع | الوصف |
|-------|-------|
| 📥 بريد وارد | خطابات تأتي من جهات خارجية |
| 📤 بريد صادر | خطابات تخرج من القسم |
| 🏖️ إجازات | طلبات الإجازات بأنواعها |
| 📋 طلبات | طلبات بأنواعها |
| 📜 قرارات داخلية | قرارات تصدر من داخل الجهاز |
| 📜 قرارات خارجية | قرارات تأتي من جهات خارجية |
| ✍️ رسائل مهمش عليها | رسائل عليها تعليمات من المدير للتنفيذ |
| 🔖 تأشيرات | موافقة أو رفض أو ملاحظة على طلب |

### بيانات كل مراسلة:
- رقم الخطاب (يُدخل يدوياً من المحفوظات)
- تاريخ الخطاب
- الجهة المرسلة / المستقبلة
- الموضوع
- اسم الموظف المعني (إن وجد)
- المرفقات (صورة من الماسح الضوئي)

### نظام تدفق العمل (Workflow):
- كل طلب أو خطاب له مسار تهميش
- مثال: موظف يقدم طلب → مدير يهمش → قانوني يهمش → مدير يقرر → مالية تنفذ
- كل شخص يرى فقط ما وصله
- تنبيه تلقائي لكل شخص عند وصول الطلب إليه
- سجل كامل لكل تهميش (من؟ متى؟ ماذا قال؟)
- تنبيه إذا توقف الطلب أكثر من X أيام
- خاص بشؤون الموظفين فقط

### متابعة ذكية:
- حالة كل خطاب: جديد / قيد المعالجة / تم الرد / مغلق
- بحث فوري بأي كلمة أو رقم أو تاريخ أو جهة
- إحصائيات: عدد الوارد والصادر / الطلبات المنجزة والمعلقة

## حادي عشر: التقارير

### تقارير الموظفين:
1. **تقرير المرشحين للترقية:** الاسم / الرقم / الدرجة الحالية / عدد العلاوات / تاريخ الاستحقاق
2. **تقرير المقبلين على التقاعد:** مع اختيار الفترة (6 أشهر / سنة / سنتين) + تنبيه من بلغ 55 سنة
3. **تقرير من تنتهي عقودهم:** مع اختيار الفترة (30 / 60 / 90 يوم)
4. **تقرير الموظفين حسب الإدارة:** متطور يشمل إحصائيات كل إدارة ومقارنة بينها

### تقارير الإجازات:
5. **تقرير رصيد الإجازات:** الرصيد الأصلي / المستخدم / المتبقي لكل موظف
6. **تقرير الموظفين في إجازة حالياً:** مع تاريخ العودة المتوقع

### تقارير العلاوات:
7. **تقرير العلاوات المستحقة هذا الشهر:** مع حقل مرشح للترقية وملاحظات (قابل للتعديل لاحقاً)
8. **تقرير تاريخ العلاوات لكل موظف:** كامل تاريخ العلاوات

### تقارير إدارية:
9. **التقرير الشهري للمدير:** ملخص شامل لكل ما حدث في الشهر
10. **التقرير السنوي للوزارة:** 5 أقسام (موظفون / إجازات / علاوات وترقيات / تقاعد وفصل ونقل / مراسلات)
11. **التقرير الربع سنوي:** الأهم والجوهر - نفس أقسام السنوي لكن لـ 3 أشهر

### مميزات جميع التقارير:
- قابلة للتخصيص (إضافة / حذف أقسام وأعمدة)
- حفظ القوالب لكل جهة
- تصدير: طباعة / PDF / Excel / Word
- حفظ نسخة من كل تقرير في الأرشيف تلقائياً

### التقرير الربع سنوي - تفاصيل:
**القسم الأول: إحصائيات الموظفين**
- إجمالي في البداية والنهاية / توزيع حسب الإدارة / نوع التعاقد / نوع الموظف / الدرجات / الذكور والإناث / من انضم ومن غادر

**القسم الثاني: الإجازات**
- إجمالي كل نوع إجازة / الأيام المستخدمة / نسبة الإجازات لكل إدارة

**القسم الثالث: العلاوات والترقيات**
- عدد من أخذوا علاوة / توزيع حسب الإدارة والدرجات / المرشحون للترقية

**القسم الرابع: التقاعد والفصل والنقل**
- إجمالي بأنواعها (تقاعد / فصل / نقل وارد وصادر وداخلي)

**القسم الخامس: المراسلات**
- إجمالي الوارد والصادر / مقسم شهرياً ونوعياً / الطلبات المنجزة والمعلقة / مقارنة بالربع السابق

- مستلمو التقرير: مدير الشؤون الإدارية / مدير مكتب المراجعة الداخلية / مدير عام الجهاز
- مقارنة ذكية مع الربع السابق
- تنبيه تلقائي عند اقتراب موعد كل ربع
